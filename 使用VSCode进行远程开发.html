# 使用 VS Code 进行远程开发

—— 将 VS Code 作为 SSH 工具，连接开发远程服务器。

# 背景

尝试了各类 SSH 工具，最终还是回到免费好用颜值又高的 VS Code. 本篇文章仅记录以供日后参考用，部分内容未详细展开说明。更多教程请参考文末链接。

VS Code 基础配置篇：VS Code 生产力指南 - 环境配置 (opens new window)

# 配置扩展

点击安装扩展：Remote - SSH (opens new window)

点击左下角 Remote 按钮即可使用。

# 其他配置

# VS Code 无法监视大型工作区的文件变化

运行此命令查看当前限制：

$ cat /proc/sys/fs/inotify/max_user_watches

编辑 /etc/sysctl.conf 文件：

$ sudo vim /etc/sysctl.conf

增加如下代码，将此限制增加到最大值：

$ fs.inotify.max_user_watches=524288

保存，启用设置：

$ sudo sysctl -p
